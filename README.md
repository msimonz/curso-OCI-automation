# OCI Automation with Terraform

Este repositorio contiene todas las actividades y ejercicios realizados durante el curso de **OCI Automation with Terraform**.

Cada carpeta y archivo está relacionado con tareas específicas cubiertas durante el curso, como la creación de recursos en Oracle Cloud Infrastructure (OCI) utilizando Terraform.

## Contenido

- **Instancia de Webserver:** Creación y configuración de servidores web en diferentes dominios de disponibilidad.
- **Balanceo de Carga:** Configuración de un balanceador de carga para distribuir el tráfico entre los servidores.
- **Seguridad:** Configuración de reglas de seguridad para permitir el acceso adecuado a los recursos.
- **Redes Virtuales:** Configuración de redes y subredes en Oracle Cloud.
